    先make,然后输入./transcoding argv[1] argv[2] argv[3]
    其中，argv[1]为文件夹名
          argv[2]可填0/1/2/3
                     0=pcm:wav -> a-lam:wav
                     1=a-lam:wav -> pcm:wav
                     2=pcm:wav -> u-lam:wav
                     3=u-lam:wav -> pcm:wav
          argv[3]为输出文件名
